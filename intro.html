h3-py¶

Python bindings for the H3 Core Library.

Installation¶

From PyPI:

pip install h3

From conda:

conda config --add channels conda-forge
conda install h3-py

Usage¶

>>> import h3
>>> lat, lng = 0, 0
>>> resolution = 0
>>> h3.geo_to_h3(lat, lng, resolution)
'8075fffffffffff'
